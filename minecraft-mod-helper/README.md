# MinecraftModHelper

Ce projet à pour but d'offrir un outil à la création et à la conception de mod Minecraft.

Il se présentera sous la forme d'un site web. L'utilisateur aura le choix de créer l'objet de son choix (bloc, item, outil, etc ...) à l'aide de simple formulaire portant sur l'ensemble des propriétés de cet objet.

Une fois le formulaire rempli et confirmé, le site génèrera le code requis à la création de l'objet et des différents fichier annexes.

Après quoi l'utilisateur n'aura plus qu'à copier coller le code dans son IDE au bon endroit.
